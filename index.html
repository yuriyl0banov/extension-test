01 — Install the preview

Load the build into Chrome

It isn't on the Web Store yet, so you'll side-load it once. Takes about a minute.

1

Unzip the build I sent you

Drop image-to-prompt.zip anywhere and extract it. Keep the folder somewhere you won't delete by accident.

2

Open the extensions page

Go to chrome://extensions and switch on Developer mode (top-right toggle).

3

Load unpacked

Click Load unpacked and pick the unzipped extension folder. The card appears with the 🖼️➜📝 icon.

4

Pin it to the toolbar

Click the puzzle-piece 🧩 in Chrome's toolbar, then the pin next to Image to prompt, so it's always one click away.

🔒
Your picture is sent securely to our service only when you ask for a prompt — there's no account, no sign-in, and no API key to set up. Works on Chrome & Edge (Chromium).
02 — Using it, frame by frame

From a picture to a prompt in 4 clicks

Same four steps every time. Here's exactly what each one looks like.

1

Find any image, anywhere

A photo, an artwork, a product shot, a screenshot — anything visible on a page works. No upload, no file picker.

Tip: it reads the actual picture on the page, so even images deep inside a gallery are fair game.

2

Right-click → “Get AI prompt”

Open the right-click menu on the image and choose Get AI prompt. That's the only command the extension adds.

Tip: if you don't see it, make sure you right-clicked directly on the image, not the empty space around it.

3

The side panel reads it

Chrome's side panel slides open on the right and shows your image dimmed while it works. Usually a couple of seconds.

What you'll see: a spinner and “Reading the image…”. Nothing to click — just wait.

4

Copy your prompt — or regenerate

Out comes a structured prompt: subject, style, mood, lighting. Hit 📋 Copy prompt and paste it into any AI image tool. Want a different take? ↻ Regenerate.

Tip: the text is fully editable after you paste — tweak tone, length, or detail to taste.

03 — Good to know

Two screens that aren't bugs

If a tester hits one of these, it's expected behaviour — not a crash.

⏳

“High demand right now”

There's a daily limit on the preview while we keep costs sane. If you hit it, prompts pause for the day — try again tomorrow.

⚠️

“Couldn't generate”

A hiccup reaching the service. Tap ↻ Try again. If it keeps happening, that's exactly the kind of thing to report.

🖼️➜📝

Empty panel

If you open the panel before using it, you'll just see a hint to right-click an image. That's the resting state.
